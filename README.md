# ExpandStar.nvim
Use star(*) command for multiple highlights  
[![asciicast](https://asciinema.org/a/494391.svg)](https://asciinema.org/a/494391)
## Requirements
* neovim >= 0.7
## Installation
```lua
-- packer
use {
    'wasden/ExpandStar.nvim'
    config = function ()
      require("expandstar").setup()
    end
  },
```
